## 👩‍💻 About Me

안녕하세요. **사용자 경험과 운영 안정성을 함께 고민하는 프론트엔드 개발자 송연지**입니다.

React와 TypeScript를 기반으로 웹 서비스를 개발하며,
현재는 Electron 기반의 보안 에이전트와 관리자 서비스를 개발하고 있습니다.

단순히 화면을 구현하는 데 그치지 않고 사용자 흐름, 실시간 통신, 장애 대응,
테스트 자동화와 문서화까지 제품이 실제 환경에서 안정적으로 동작하는 과정을 고민합니다.

```text
사용자가 이해하기 쉬운 흐름을 설계합니다.
반복되는 작업은 자동화합니다.
문제가 반복되면 코드가 아니라 구조부터 다시 살펴봅니다.
```
